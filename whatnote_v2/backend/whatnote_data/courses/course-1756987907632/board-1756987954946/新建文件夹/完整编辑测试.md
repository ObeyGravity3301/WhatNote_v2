# 完整编辑功能测试

这个版本支持完整的编辑操作！

## 新特性

- **完整编辑**：支持backspace、delete、粘贴换行文本
- **代码块渲染**：多行代码块正确显示背景
- **透明叠加**：编辑器在上层，预览在下层
- **滚动同步**：编辑和预览同步滚动

## 代码块测试

```python
def hello_world():
    print("Hello, World!")
    
    # 这是一个多行代码块
```
